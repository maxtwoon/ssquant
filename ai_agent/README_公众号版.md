## 02 三大核心功能

### 🎯 功能一：AI写策略

只需要告诉 AI 你想要什么：

```
"写一个海龟突破策略，20日突破入场"
"给我一个MACD + RSI组合策略"  
"双均线金叉做多，死叉做空"
```

几秒钟，完整策略代码就出来了。

---

### 🎯 功能二：一键回测

写完就能跑！

- 点击运行按钮
- 实时查看回测日志
- 自动生成可视化报告

**支持的回测指标**：
- 📈 年化收益率
- 📉 最大回撤
- ⚡ 夏普比率
- 🎯 胜率
- 💰 盈亏比

---

### 🎯 功能三：自动优化（AUTO模式）

这个功能才是真正的**杀手锏**！

1️⃣ AI 生成策略  
2️⃣ 自动运行回测  
3️⃣ AI 分析报告

**真正的"全自动驾驶"**，你只需要喝杯咖啡等结果。

## 06 常见问题

### ❓ 提示找不到 ssquant 模块？

确保目录结构正确：

```
ssquant-main/
├── ai_agent/     ← AI Agent 在这
└── ssquant/      ← 框架也要在
```

### ❓ AI 不响应？

- 检查 API Key 是否正确
- 检查网络是否正常
- 查看控制台错误信息

### ❓ 回测没输出？

检查策略代码是否包含完整的运行器配置。
